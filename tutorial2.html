Tutorial forward IP Local ke VPN 

Kita akan menggunakan port SSH mikrotik untuk dijadikan port yang mengarah ke halaman web Access Point 

jadi pastikan port SSH itu aktif 

Setelah anda login mikrotik via winbox , masuk ke IP - Firewall - NAT

Tambahkan Rule NAT baru 

- Chain dipilih Dst-nat 

- Src address diisi 10.20.20.1 ( IP Server VPN )

- Protocol diisi TCP 

- Port 22 ( Sesuai Port IP Service  SSH) 

Lalu pindah ke tab action , action pilih dst-nat, to addresses di isi

IP Akses Point ( 192.168.5.3 ) 
To ports diisi 80 

Isi dengan IP local anda yang ingin di forward ke publik

Setelah itu coba akses IP Publik dengan PORT yang diberikan admin .
