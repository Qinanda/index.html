Tutorial Install VPN Remote di Mikrotik

Note  : Pastikan port service mikrotik default

Winbox : 8291 , Api : 8728
FTP 21, SSH :  22
Web : 80

Pertama buat OVPN Client di Interface - Add - OVPN Client 

Masukkan IP Server dan username password yang diberikan admin dan sesuaikan sesuai gambar diatas 

pastikan status : connected 

Buat netwatch di Tools - Netwatch 

Masukkan IP Host 10.20.20.1 
Jangan diganti .

Setelah itu admin akan memberikan IP dan Port Remote mikrotik  

silahkan login winbox seperti gambar berikut ,

Contoh IP & Port yang diberikan adalah id2.qishiro.xyz dengan port 2500 untuk mengakses winbox
